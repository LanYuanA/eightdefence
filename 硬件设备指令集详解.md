# 八防系统硬件设备指令集详解

本文档基于 `EquipmentConstant.java`整理，包含了系统中所有接入硬件设备的通信地址、指令格式以及具体的读写指令。所有十六进制指令在实际下发至设备前，需注意根据相应协议补全校验码（常规设备通过 `getInstruction()` 追加 CRC16 校验码，新风机通过 `getInstruction2()` 对 256 取余作为校验位）。

---

## 01. 室内空气质量变送器（云测仪）

- **设备标识 (TAG_GROUP1)**: `01`
- **设备地址**: `30`
- **读指令起始**: `3003` (地址 `30` + 功能码 `03`)
- **指令结尾**: `0001`

### 数据读取指令 (InstructionGroup01)

| 监测项目                  | 指令拼接格式 (起始 + 寄存器地址 + 结尾) | 完整基础指令         | 字段标识 (TAG) |
| :------------------------ | :-------------------------------------- | :------------------- | :------------- |
| **PM2.5** (μg/m³) | 3003 + 0000 + 0001                      | `300300000001802B` | PM25           |
| **PM10** (μg/m³)  | 3003 + 0001 + 0001                      | `300300010001D1EB` | PM10           |
| **湿度** (%RH)      | 3003 + 0002 + 0001                      | `30030002000121EB` | HUMIDITY       |
| **温度** (℃)       | 3003 + 0003 + 0001                      | `300300030001702B` | TEMPERATURE    |
| **TVOC** (ppb)      | 3003 + 0007 + 0001                      | `30030007000131EA` | TVOC           |
| **甲醛** (ppm)      | 3003 + 0009 + 0001                      | `3003000900015029` | CH2O           |
| **臭氧/O3** (ppm)   | 3003 + 000A + 0001                      | `3003000A0001A029` | O3             |
| **二氧化氮** (ppm)  | 3003 + 000F + 0001                      | `3003000F0001B028` | CO2            |
|                           |                                         |                      |                |

## 03. 驱鼠器 (MouseTrapAlarm)

- **设备标识 (TAG_GROUP4)**: `04`
- **设备地址**: `80`

### 控制与读取指令 (InstructionGroup04)

| 功能                     | 发送指令(TX)         | TAG         | 备注说明                                        |
| :----------------------- | :------------------- | :---------- | :---------------------------------------------- |
| **读取继电器状态** | `8001000000011BFD` | MouseStatus | 返回数据的最后一位:`01`为接通，`00`为未接通 |
| **继电器吸合(开)** | `80050000FF0089D2` | MouseOpne   | 成功则原路返回，失败返回空                      |
| **继电器断开(关)** | `800500000000C822` | MouseClose  | 成功则原路返回，失败返回空                      |

## 05. 红外探测器

- **设备标识 (TAG_GROUP5)**: `05`
- **设备地址**: `40`
- **读指令格式**: 起始 `4003` + 结尾 `0001`

### 指令

返回数据 `0000` 表示无人，`0001` 表示有人。

| 功能                   | 读取指令             | 字段标识 (TAG) |
| :--------------------- | :------------------- | :------------- |
| **红外感应状态** | `4003000600016B1A` | INFRARED_STATE |
| **雷达感应状态** | `4003000700013ADA` | RADAR_STATE    |
